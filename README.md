# PromptHub

ChatGPT에서 자주 쓰는 이미지 생성 프롬프트를 저장하고 빠르게 삽입하는 개인용 Chrome 확장 프로그램입니다.

## 현재 MVP 기능

- ChatGPT 오른쪽에 PromptHub 사이드바 주입
- 프롬프트 추가, 수정, 삭제
- 썸네일 이미지 등록
- 제목, 설명, 태그, 본문 검색
- 즐겨찾기 우선 정렬
- `사용하기` 버튼으로 ChatGPT 입력창에 프롬프트 삽입
- `$키워드` 자동완성으로 입력 중 프롬프트 치환
- 자동 삽입 실패 시 클립보드 복사 fallback
- JSON 백업 내보내기/가져오기

## 개발

```bash
npm ci
npm run dev
```

## 확장 프로그램 빌드

```bash
npm run build
```

빌드 결과는 `dist/`에 생성됩니다.

## Chrome에서 로드

1. `npm run build`를 실행합니다.
2. Chrome에서 `chrome://extensions`를 엽니다.
3. 오른쪽 위 `개발자 모드`를 켭니다.
4. `압축해제된 확장 프로그램을 로드합니다`를 누릅니다.
5. 이 레포의 `dist/` 폴더를 선택합니다.
6. `https://chatgpt.com`에서 PromptHub 사이드바가 뜨는지 확인합니다.

## 사용법

- 사이드바의 `추가` 버튼으로 자주 쓰는 이미지 스타일 프롬프트를 저장합니다.
- 프롬프트를 선택한 뒤 `사용하기`를 누르면 ChatGPT 입력창에 삽입됩니다.
- ChatGPT 입력창에서 `$ani`처럼 입력하면 저장된 프롬프트 자동완성이 열립니다.
- `내보내기`로 JSON 백업을 저장하고, `가져오기`로 다시 합칠 수 있습니다.
